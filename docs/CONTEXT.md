# 全国退休计算器 - 项目上下文

## 📋 项目概述
一个支持全国各省份、包含渐进式延迟退休政策的网页版退休计算工具。

## 🎯 核心目标
- 帮助用户计算自己的法定退休年龄（含延迟退休）
- 估算养老金待遇
- 提供政策说明和参考数据

## 🛠️ 技术栈
- **前端**: 原生 HTML5 + CSS3 + ES6 JavaScript
- **单文件架构**: 所有代码合并到一个 HTML 文件（便于分享和部署）
- **无依赖**: 不引入外部框架，减少加载时间
- **响应式**: 支持桌面端和移动端

## 🧮 核心计算逻辑

### 1. 渐进式延迟退休（2025年起实施）

**男性（原60岁退休）**
- 1965年前出生：60岁（不受延迟影响）
- 1965-1976年出生：每4个月出生时间延迟1个月
- 1976年后出生：63岁（延迟满36个月）

**女性干部（原55岁退休）**
- 1970年前出生：55岁（不受延迟影响）
- 1970-1981年出生：每4个月出生时间延迟1个月
- 1981年后出生：58岁（延迟满36个月）

**女性工人（原50岁退休）**
- 1975年前出生：50岁（不受延迟影响）
- 1975-1985年出生：每2个月出生时间延迟1个月
- 1985年后出生：55岁（延迟满60个月）

### 2. 养老金计算公式

```
基础养老金 = (全省上年度职工月平均工资 + 本人指数化月平均缴费工资) ÷ 2 × 缴费年限 × 1%

个人账户养老金 = 个人账户储存额 ÷ 计发月数

合计月养老金 = 基础养老金 + 个人账户养老金
```

**计发月数参考**:
- 60岁退休：139个月
- 55岁退休：170个月
- 50岁退休：195个月

## ⚠️ 已知限制
1. 延迟退休政策细则可能有调整，需跟进官方最新文件
2. 养老金计算未考虑过渡性养老金（针对有视同缴费年限的人员）
3. 各省份计发基数每年调整，数据需定期更新

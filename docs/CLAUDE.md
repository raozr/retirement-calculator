# CLAUDE.md - 全国退休计算器

> 本文件供 AI 助手快速理解项目上下文和开发规范。

## 项目概述

**全国退休计算器** - 一个支持全国各省份、包含渐进式延迟退休政策的网页版退休计算工具。

- **版本**: v1.0.0
- **技术栈**: 原生 HTML5 + CSS3 + ES6，单文件架构
- **目标用户**: 需要计算退休年龄和估算养老金的普通用户
- **数据持久化**: LocalStorage 自动保存用户输入

## 核心功能

### 1. 退休年龄计算
- 支持渐进式延迟退休（2025政策），精确到月
- 区分男性、女性干部、女性工人/灵活就业人员
- **特殊工种提前退休**（男55岁/女45岁，需满8-10年工龄）
- **病退提前退休**（男50岁/女45岁，完全丧失劳动能力）

### 2. 养老金测算
- 基础养老金计算（各省份计发基数不同）
- 个人账户养老金计算（按退休年龄对应计发月数）
- **过渡性养老金**（针对1996年前参加工作的"中人"）
- **缴费基数验证**（自动校验是否在省份上下限范围内）
- **养老金预测**（考虑年度调整率，预测未来30年）
- 支持北京、上海、广东、江苏、浙江、山东等省份

### 3. 数据持久化
- LocalStorage 自动保存所有表单输入
- 页面刷新后自动恢复数据
- 重置按钮清除保存的数据

### 4. 导出功能
- 支持导出退休计算结果为 txt 文件
- 支持导出养老金测算结果为 txt 文件

## 渐进式延迟退休政策（2025年起实施）

### 男性（原60岁 → 延迟至63岁）
- 1965年前出生：60岁（不受延迟影响）
- 1965-1976年出生：每4个月延迟1个月
- 1976年后出生：63岁（延迟满36个月）

```javascript
const retirementConfig = {
    male: { startAge: 60, startYear: 1965, maxAge: 63, interval: 4, maxDelay: 36 }
};

// 延迟月数计算
const monthsSinceStart = (birthYear - 1965) * 12 + (birthMonth - 1);
const delayMonths = Math.floor(monthsSinceStart / 4);  // 最大36个月
```

### 女性干部（原55岁 → 延迟至58岁）
- 1970年前出生：55岁（不受延迟影响）
- 1970-1981年出生：每4个月延迟1个月
- 1981年后出生：58岁（延迟满36个月）

### 女性工人/灵活就业（原50岁 → 延迟至55岁）
- 1975年前出生：50岁（不受延迟影响）
- 1975-1985年出生：每2个月延迟1个月
- 1985年后出生：55岁（延迟满60个月）

### 特殊工种提前退休
- 男性满55周岁，女性满45周岁
- 工作年限要求：
  - 繁重体力劳动：满10年
  - 高空作业：满10年
  - 井下/高温作业：满9年
  - 有毒有害工作：满8年

### 病退提前退休
- 因病或非因工致残，完全丧失劳动能力
- 男性满50周岁，女性满45周岁
- 养老保险累计缴费满15年

## 养老金计算公式

### 基础养老金
```
基础养老金 = (全省上年度职工月平均工资 + 本人指数化月平均缴费工资) ÷ 2 × 缴费年限 × 1%
```

### 个人账户养老金
```
个人账户养老金 = 个人账户储存额 ÷ 计发月数
```

### 过渡性养老金（1996年前参加工作）
```
过渡性养老金 = 全省上年度职工月平均工资 × 平均缴费指数 × 视同缴费年限 × 1.3%
```

### 关键算法

#### 延迟月数计算
```javascript
function calculateDelayMonths(birthYear, birthMonth, config) {
    if (birthYear < config.startYear) {
        return 0;  // 延迟前出生，不延迟
    }
    const monthsSinceStart = (birthYear - config.startYear) * 12 + (birthMonth - 1);
    if (monthsSinceStart <= 0) {
        return 0;  // 起始月延迟0个月
    }
    let delayMonths = Math.floor(monthsSinceStart / config.interval);
    return Math.min(delayMonths, config.maxDelay);  // 不超过最大延迟
}
```

## 开发注意事项

### 1. 日期处理
- 闰年2月29日处理：退休日期计算时需检查，非闰年退到2月28日
- 退休日期 = 出生日期 + 退休年龄（年月）

### 2. 边界条件
- 1965-01、1970-01、1975-01 分别是三类人群的延迟起点，延迟月数为0
- 缴费年限不足15年时要提示用户无法领取养老金
- 缴费基数超出省份上下限时要给出警告提示

### 3. 浮点数精度
- 使用 toFixed(2) 格式化金额
- 比较时使用 tolerance（容差）进行比较

### 4. LocalStorage
- 每次输入变化自动保存（onchange 事件）
- 页面加载时自动恢复（DOMContentLoaded）
- 重置按钮调用 localStorage.removeItem()

### 5. 导出功能
- 使用 Blob 和 URL.createObjectURL 生成下载链接
- 导出内容为纯文本格式（.txt）

## 常见错误

1. **延迟月数计算偏差**：1965-01应延迟0个月，不是1个月
2. **女性身份类型混淆**：干部55岁起延，工人50岁起延
3. **计发月数固定**：应根据退休年龄动态获取（45岁216，50岁195，55岁170，60岁139）
4. **特殊工种最低年龄**：男性55岁、女性45岁是硬性门槛
5. **缴费基数验证遗漏**：需要校验是否在省份规定的上下限范围内
6. **LocalStorage键名冲突**：确保键名唯一，避免与其他应用冲突

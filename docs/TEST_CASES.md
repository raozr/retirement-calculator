## 🔧 测试工具使用说明

### 方式一：浏览器可视化测试

```bash
# 打开测试页面
open src/test.html
```

### 方式二：控制台快速测试

```bash
# 在浏览器控制台执行
runAllTests()
```

### 方式三：人工界面测试

1. **延迟退休计算**:
   - 选择不同性别和出生日期
   - 验证延迟月数计算是否正确
   - 测试特殊工种和病退选项

2. **养老金测算**:
   - 选择不同省份验证计发基数
   - 输入超出范围的缴费基数验证提示
   - 勾选过渡性养老金验证展开

3. **数据持久化**:
   - 填写表单后刷新页面
   - 验证数据是否正确恢复

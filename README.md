# 全国退休计算器

一个支持全国各省份、包含渐进式延迟退休政策的网页版退休计算工具。

![版本](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## ✨ 功能特性

- 📅 **退休年龄计算**：支持渐进式延迟退休（2025政策），精确到月
- 💰 **养老金测算**：基础养老金 + 个人账户养老金 + 过渡性养老金
- 🏭 **特殊工种**：支持提前退休计算（高空、高温、井下、有毒有害等）
- 🏥 **病退支持**：因病/非因工致残提前退休计算
- 📊 **养老金预测**：考虑年度调整率，预测未来30年养老金
- 💾 **数据持久化**：LocalStorage 自动保存，刷新不丢失
- 📱 **响应式设计**：桌面端和移动端都能正常使用
- 🎨 **瑞士风格**：国际主义设计风格，清晰专业
- 📖 **政策说明**：内置延迟退休政策和各省份数据参考
- 📤 **导出功能**：支持导出计算结果为 txt 文件

## 📋 使用指南

### 1. 计算退休年龄
1. 选择性别和出生日期
2. 选择所在省份和身份类型（男性职工/女性干部/女性工人/灵活就业）
3. 选择退休类型（正常/特殊工种/病退）
4. 点击"计算退休信息"

### 2. 测算养老金
1. 切换到"养老金测算"标签
2. 选择省份（自动填充计发基数和缴费上下限）
3. 输入缴费基数、缴费年限、个人账户余额
4. 如适用，勾选"视同缴费年限"计算过渡性养老金
5. 点击"计算养老金"

## 🧮 计算说明

### 渐进式延迟退休（2025年起实施）

**男性**（原60岁 → 延迟至63岁）
- 1965年前出生：60岁
- 1965-1976年出生：每4个月延迟1个月
- 1976年后出生：63岁

**女性干部**（原55岁 → 延迟至58岁）
- 1970年前出生：55岁
- 1970-1981年出生：每4个月延迟1个月
- 1981年后出生：58岁

**女性工人**（原50岁 → 延迟至55岁）
- 1975年前出生：50岁
- 1975-1985年出生：每2个月延迟1个月
- 1985年后出生：55岁

### 特殊工种提前退休
- **年龄要求**：男性满55周岁，女性满45周岁
- **工作年限**：
  - 繁重体力劳动：满10年
  - 高空作业：满10年
  - 井下/高温作业：满9年
  - 有毒有害工作：满8年

### 病退提前退休
- **条件**：因病或非因工致残，完全丧失劳动能力
- **年龄**：男性满50周岁，女性满45周岁
- **缴费**：养老保险累计缴费满15年

### 养老金计算公式

```
基础养老金 = (全省上年度职工月平均工资 + 本人指数化月平均缴费工资) ÷ 2 × 缴费年限 × 1%

个人账户养老金 = 个人账户储存额 ÷ 计发月数

过渡性养老金 = 全省上年度职工月平均工资 × 平均缴费指数 × 视同缴费年限 × 1.3%
```

**计发月数参考**：45岁(216) 50岁(195) 55岁(170) 60岁(139) 63岁(117)

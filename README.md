# 全国退休计算器

一个支持全国各省份、包含渐进式延迟退休政策的网页版退休计算工具。

![版本](https://img.shields.io/badge/version-0.3.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## ✨ 功能特性

- 📅 **退休年龄计算**：支持渐进式延迟退休（2025政策），精确到月
- 💰 **养老金测算**：基础养老金 + 个人账户养老金，多省份数据
- 🏭 **特殊工种**：支持提前退休计算
- 📱 **响应式设计**：桌面端和移动端都能正常使用
- 🎨 **精美界面**：深色渐变主题，卡片式布局
- 📖 **政策说明**：内置延迟退休政策和各省份数据参考

## 🚀 快速开始

### 方式一：直接打开（推荐）
```bash
双击打开 src/index.html
```

### 方式二：本地服务器
```bash
cd retirement-calculator

```

## 📋 使用指南

### 1. 计算退休年龄
1. 选择性别和出生日期
2. 选择所在省份和身份类型
3. （可选）填写特殊工种信息
4. 点击"计算退休信息"

### 2. 测算养老金
1. 切换到"养老金测算"标签
2. 选择省份（自动填充计发基数）
3. 输入缴费基数、年限、个人账户余额
4. 点击"计算养老金"

## 🧮 计算说明

### 渐进式延迟退休（2025年起实施）

**男性**（原60岁 → 延迟至63岁）
- 1965年前出生：60岁
- 1965-1976年出生：每4个月延迟1个月
- 1976年后出生：63岁

**女性干部**（原55岁 → 延迟至58岁）
- 1970年前出生：55岁
- 1970-1981年出生：每4个月延迟1个月
- 1981年后出生：58岁

**女性工人**（原50岁 → 延迟至55岁）
- 1975年前出生：50岁
- 1975-1985年出生：每2个月延迟1个月
- 1985年后出生：55岁

### 养老金计算公式

```
基础养老金 = (全省上年度职工月平均工资 + 本人指数化月平均缴费工资) ÷ 2 × 缴费年限 × 1%

个人账户养老金 = 个人账户储存额 ÷ 计发月数（60岁退休为139个月）
```
